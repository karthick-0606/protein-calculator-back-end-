Protein Calculator – Backend
This is the Spring Boot backend for the Protein Calculator application.
It handles user data storage, protein calculation logic, and CRUD operations for the application.

🚀 Live Backend URL (Render – Free Plan)
👉 https://protein-calculator-back-end.onrender.com/api/protein
